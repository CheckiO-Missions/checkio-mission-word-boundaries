Otrzymuje Pan ciąg znaków (words) z kilkoma słowami oddzielonymi spacjami oraz liczbę całkowitą (n) - indeks symbolu w tym ciągu. Proszę wypisać słowo (część words, oddzielone spacjami), które zawiera symbol pod tym indeksem. Jeśli żaden element nie pasuje do tego indeksu, proszę zwrócić {% if is_js %}null{% else %}None{% endif %}.

Proszę zauważyć: Indeks ma wartość 0 i odnosi się do pozycji znaku w ciągu znaków (w tym znaków spacji).

Dane wejściowe:
  • Liczba całkowita "n" - indeks symbolu w słowach.
  • Łańcuch "words" z niektórymi elementami oddzielonymi spacjami.

    • Wyjście: Pojedynczy wiersz - element zawierający symbol pod tym indeksem.

    {{init_code_tmpl}}

    Warunki wstępne:

  • 0 < n < 30
  • 0 < długość elementów < 30
