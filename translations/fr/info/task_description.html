On vous donne une chaîne (words) avec quelques mots séparés par des espaces et un entier (n) - l'index d'un symbole dans cette chaîne. Vous devez imprimer le mot (partie de words, délimitée par des espaces) qui contient le symbole sous cet index. Si aucun élément ne correspond à cet index, vous devez renvoyer {% if is_js %}null{% else %}None{% endif %}.

Remarque: l'indice est basé sur 0 et se réfère à la position d'un caractère dans la chaîne: L'indice est basé sur 0 et se réfère à la position d'un caractère dans la chaîne (y compris les caractères d'espacement).

Entrée:
  • Un entier "n" - indice d'un symbole dans les mots.
  • Une chaîne de caractères "mots" dont certains éléments sont séparés par des espaces.

    • Résultat: Une seule ligne - élément contenant le symbole sous cet index.

    Exemples: 
    {{init_code_tmpl}}

    Conditions préalables:

  • 0 < n < 30
  • 0 < longueur des éléments < 30
